## Atlassian Bamboo Details

### The Meta should be configured as follows:
```
{
  "user": "User Bamboo",
  "key": "User Key",
  "bambooServer": "Server IP Address",
  "projects": [
    {
      "name":"project"
    }
  ]
}
```
#### Provider name: bamboo

The supported metrics are:
- build
- release
